This folder archives an exploratory SiPM distance-analysis attempt.

Contents:
- `tengiz_distance_analysis.ipynb`
- `geometry_check.ipynb`
- `integral_vs_R.ipynb`
- `tengiz_output.csv`
- `mu10_0.csv`

Why it is archived here:
- The work was useful as a scratch investigation, but it did not lead to a clean or convincing result.
- The assumptions were too brittle to treat the outputs as part of the main analysis flow.
- The files are being kept for reference so the attempt is not lost, but they should be treated as non-authoritative.

Notes:
- The notebooks and CSV files were moved together so their local file references still make sense within this folder.
- If this line of work is revisited later, it should probably be rebuilt from a cleaner starting point rather than extended in place.
